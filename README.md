تطبيق ويب تقدمي (PWA) ذكي، خفيف وسريع، يعمل بالكامل **بدون الحاجة لاتصال بالإنترنت**. يهدف إلى مساعدة الركاب في التخطيط لرحلاتهم داخل شبكة النقل المتطورة في القاهرة الكبرى، والتي تشمل خطوط المترو الثلاثة، مونوريل شرق النيل، والقطار الكهربائي الخفيف (LRT).

---

## ✨ ميزات التطبيق الرئيسية

* 🗺️ **تغطية شاملة للشبكة المحدثة**: يدعم خطوط المترو الثلاثة (بما في ذلك تفرعات الخط الثالث إلى جامعة القاهرة ومحور روض الفرج)، إلى جانب مونوريل شرق النيل والقطار الكهربائي الخفيف (LRT).
* 🧮 **حساب ذكي للمسارات**: يعتمد على نسخة مطورة من خوارزمية **دايجسترا (Dijkstra)** تفرض "عقوبة وزن" على محطات التبديل لتجنب كثرة التحويلات غير المريحة وإعطاء المسار الأكثر سلاسة للمستخدم.
* 🔎 **بحث مرن وذكي**: يتجاهل علامات التشكيل، ويوحد الحروف المتشابهة (مثل أ، إ، آ، هـ، ة، ي، ى)، كما يدعم المترادفات والأسماء الشائعة (مثل "رمسيس" للوصول إلى "الشهداء"، و"التحرير" للوصول إلى "السادات").
* 💵 **تقدير دقيق للتكلفة والوقت**: يحسب أسعار التذاكر الإجمالية تلقائياً بناءً على عدد محطات كل وسيلة مواصلات، مع تقديم زمن تقديري منطقي للرحلة.
* 💾 **سجل الرحلات الأخيرة**: يحفظ آخر 5 رحلات بحث فريدة محلياً باستخدام `localStorage` للوصول السريع إليها بضغطة زر واحدة.
* 📱 **تصميم عصري متجاوب**: واجهة زجاجية (Glassmorphism) مريحة للعين، متوافقة بالكامل مع الشاشات الصغيرة وأجهزة الآيفون والأندرويد والشاشات الكبيرة.
* 🌐 **دعم التشغيل بدون إنترنت (PWA)**: جاهز للتثبيت كبرنامج مستقل على شاشتك الرئيسية ويعمل بفعالية تامة دون إنترنت.

## 🛠️ كواليس التطوير والبرمجة

### خوارزمية التوجيه (Routing Engine)
لتفادي توجيه الراكب عبر مسارات معقدة تحتوي على تحويلات متعددة غير ضرورية، تم تطوير منطق التوجيه لفرض عقوبة منطقية (Transfer Penalty) أثناء حساب الوزن بين المحطات:

```javascript
// مقتطف من منطق البحث المطور لمنع التبديل غير المريح
if (currentInfo.lastLine && currentInfo.lastLine !== edgeLine) {
    stepCost += 12; // فرض وزن إضافي لمنع كثرة التحويلات إلا للضرورة القصوى
}
```

### معالجة النصوص والبحث
تمت برمجة التطبيق ليكون مرناً مع لغة المستخدم اليومية عبر إزالة التشكيل وتحليل الاختصارات الشائعة:
```javascript
const stripDiacritics = (s) => {
    return s.trim()
        .replace(/[أإآأ]/g, 'ا')
        .replace(/ة/g, 'ه')
        .replace(/[ىي]/g, 'ي')
        .replace(/[\u064B-\u065F]/g, '');
};
```

## 📦 ملفات المشروع الموصى بها على المستودع

لضمان عمل التطبيق كـ **PWA** متكامل وقابل للتثبيت، يفضل أن يحتوي مستودعك على الهيكل التالي:

```text
├── index.html          # كود التطبيق الأساسي (الواجهة والمنطق)
├── manifest.json       # ملف تعريف التطبيق التقدمي
├── sw.js               # الـ Service Worker المسؤول عن التخزين المؤقت والتشغيل دون إنترنت
├── favicon-32.png      # أيقونة المتصفح والتطبيق (32x32)
└── apple-touch-icon.png# أيقونة الشاشة الرئيسية لأجهزة Apple (180x180)
```

## 🚀 التشغيل والتثبيت المحلي

يمكنك تشغيل المشروع ببساطة وبدون الحاجة لأي خوادم معقدة:

1. **نسخ المستودع (Clone):**
   ```bash
   git clone https://github.com/seif4d/Metro.git
   ```
2. **تشغيل المشروع:**
   * يمكنك فتح ملف `index.html` مباشرة في أي متصفح.
   * لتفعيل ميزات الـ PWA والتثبيت، يُفضل تشغيله عبر خادم محلي (Local Server) مثل إضافة Live Server في VS Code، أو عبر استضافة مجانية مثل GitHub Pages.

## 🎫 تسعير التذاكر المدمج في التطبيق (تحديث 2026)

* **مترو القاهرة**:
  - من 1 إلى 9 محطات: **10 جنيهات**
  - من 10 إلى 16 محطة: **12 جنيهاً**
  - من 17 إلى 23 محطة: **15 جنيهاً**
  - أكثر من 23 محطة: **20 جنيهاً**
* **مونوريل شرق النيل**: يبدأ من **20 جنيهاً** وحتى **80 جنيهاً** حسب عدد المحطات.
* **القطار الكهربائي الخفيف (LRT)**: يبدأ من **10 جنيهات** وحتى **20 جنيهاً**.
